#视频推荐系统
	视频智能推荐系统可以获取多家在线视频网站提供的视频内容信息，并对相
	关视频内容进行分析，基于用户个性化推荐，根据每个用户的兴趣、年龄等多个
	维度进行个性化推荐。推荐内容不仅包括狭义上的电影、电视剧，还可以包括音
	乐、新闻、游戏、教育、旅游、购物等多种与视频相关的资源，为用户提供便利。
